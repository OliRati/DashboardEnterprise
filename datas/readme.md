# FOAD PHP Entreprise

Pour la **FOAD** du jour vous aller de voir créer un **C.R.U.D** , c'est à dire mettre en place des formulaires et des pages de traitement en **PHP** pour gérer la table  **employes** de la base de donnée **entreprise**. 

La base de donnée sur laquelle vous aller travailler est la suivante [entreprise](entreprise.sql).

**C.R.U.D** :
- **C**reate , formulaire pour **créer** un nouvel employé
- **R**read , page qui **liste** tous les employés
- **U**pdate , formulaire pour **mettre à jour** un employé
- **D**delete , formulaire pour **supprimer** un employé

Bien prendre en compte la **sécurité**, ne **JAMAIS** faire confiance à ce que va entrer l'utilisateur donc bien nettoyer les champs de formulaires.

Pour votre **dashboard** vous êtes libres d'utiliser une librairie **CSS** ou de créer la votre.
